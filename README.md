# Antonin Deburge — Site Avocat

Site vitrine pour Maître Antonin Deburge, Avocat au Barreau de Paris.

## Stack

- HTML5 / CSS3 / JavaScript vanilla
- Police : Cormorant Garamond + Montserrat (Google Fonts)
- Déployé via Netlify (lié à ce dépôt)

## Structure

```
/
└── index.html    ← Site complet (single page)
```

## Design

Direction artistique : luxe minimaliste extrême
- Fond ivoire `#faf8f5`
- Typographie fine, très espacée (letter-spacing élevé)
- Filets 1px comme unique décoration
- Palette : ivoire + brun chaud `#2a2218` + beige `#c8b8a8`
- Animations de fondu lent au scroll (Intersection Observer)
- Zéro image, zéro icône, zéro bouton flashy

## Déploiement

Push sur `main` → Netlify déploie automatiquement.
